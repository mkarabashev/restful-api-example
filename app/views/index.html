Microservice API

Add a unix timestamp or the date to the url

Use now or don't give a parameter to get the current time

Examples:

/api?time

/api?time=now

/api?time=1450137600

/api?time=December%2015,%202015

Outcome:

{"unix":1450137600,"natural":"December 15, 2015"}

Add whoami to get useragent data

Examples:

/api?whoami

Outcome:

{"ip":"::1","language":"bg,en-US","os":"Windows 10.0"}

Shorten a URL

Examples:

/api?url=www.google.com

Outcome:

{"url": {"longUrl": "www.google.com", "shortUrl": "localhost:3000/2"}
